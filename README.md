## Usage

After cloning 
1. `sh symlink.sh`
2. Install oh-my-zsh (see https://ohmyz.sh/#install)
3. `cp mymindwentblvnk.zsh-theme ~/.oh-my-zsh/themes`

## For vim

1. Install Vundle with `git clone https://github.com/VundleVim/Vundle.vim.git ~/.vim/bundle/Vundle.vim`
2. Open Vim
3. `:PluginInstall`
